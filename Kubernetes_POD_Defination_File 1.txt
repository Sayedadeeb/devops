
+++++++++++++++++++++++++++++ POD DEFINITION FILE +++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

Ex1:  Create a pod definition file to start nginx in a pod. 
Name the pod as nginx-pod, name the container as appserver. 




$ vim pod-definition1.yml

---
apiVersion: v1
kind: Pod
metadata:
 name: nginx-pod
 labels:
  author: sunil
  type: reverse-proxy
spec:
 containers: 
  - name: appserver
    image: nginx

:wq

$ kubectl create -f pod-definition1.yml
